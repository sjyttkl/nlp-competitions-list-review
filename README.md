# nlp-competitions-list-review

### 目的

建立一个**只专注于NLP比赛**的方案分享讨论的项目，讨论时期不限，可包括赛前分析，赛中讨论，赛后复盘，其中以赛后复盘为主。

### 内容

讨论比赛平台，赛题背景，数据格式，主要思路，技术实现，模型设计，框架选择，评估指标等方面梳理NLP比赛。
